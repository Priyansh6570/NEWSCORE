A type-led, multilingual news system. Centred mastheads, hairline-ruled typographic grids, a two-tempo front page — and a script-resilient type stack that renders the same layout cleanly across Latin, Devanagari, Tamil, Bengali and Urdu.

An editorial serif voice for journalism, a clean sans for the interface — each chosen for genuine multi-script coverage. Headlines are never locked to one script's density.

Grid · rhythm

A 12-column field on an 8px spacing scale; restraint over ornament. Cards are replaced by hairline rules and whitespace. Radii stay small — pills for chips, gentle corners for bands.
