Six skins, one layout

The home-A layout is identical in every frame — same grid, hairline rules, masthead, fast-lane wire, story grid and dark spotlight. Only the skin changes: paper, ink, hairline weight and the single accent (and its LIVE state). Each direction is shown desktop-light in English, desktop-dark in a non-Latin script, and mobile — so you can judge the accent where it matters: the LIVE pulse, the hairlines, the byline tags and links. Light + dark tokens are defined for all six.
